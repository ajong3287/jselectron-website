## 학습 내용

### PDF 처리 핵심 명령어
```bash
# 이미지 추출 (JPEG 형식)
pdfimages -j input.pdf output_prefix

# 이미지 최적화 (macOS)
sips -Z [max_size] -s formatOptions [quality] input.jpg --out output.jpg

# 이미지 크기 확인
ls -lh *.jpg | awk '{print $5, $9}'
```
