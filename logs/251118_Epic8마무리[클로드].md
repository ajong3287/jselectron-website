## 🔧 기술적 세부사항

### 디자인 시스템 통일

**섹션 배경 패턴**:
```tsx
// 패턴 1: 파란색 그라데이션 헤더
className="bg-gradient-to-r from-blue-600 to-blue-800 text-white py-16 px-6 text-center"

// 패턴 2: 흰색 배경 컨텐츠
className="bg-white py-16 px-6"

// 패턴 3: 연한 파란색 배경
className="bg-blue-50 py-16 px-6"
```

**카드 디자인**:
```tsx
className="p-6 bg-white border-2 border-blue-100 rounded-xl shadow-lg
           hover:shadow-2xl hover:border-blue-500 transition-all"
```

**버튼 디자인**:
```tsx
// Primary CTA
className="inline-block text-white bg-blue-600 hover:bg-blue-700
           px-8 py-3 rounded-lg text-lg font-bold shadow-lg
           transition-all transform hover:scale-105"

// Secondary CTA
className="inline-block text-blue-700 bg-white border-2 border-blue-600
           hover:bg-blue-600 hover:text-white px-8 py-3 rounded-lg
           text-lg font-bold shadow-lg transition-all"
```

### 중앙 정렬 시스템
```tsx
// 넓은 컨텐츠 영역 (제품 그리드, 핵심 가치)
<div className="max-w-6xl mx-auto">

// 중간 컨텐츠 영역 (본문, 설명)
<div className="max-w-4xl mx-auto">

// 좁은 컨텐츠 영역 (폼, 검색)
<div className="max-w-3xl mx-auto">
```
